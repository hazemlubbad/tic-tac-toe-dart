# 🕹️ Dart Console Tic-Tac-Toe Game

A simple 2-player Tic-Tac-Toe game built using Dart for the console. Perfect for learning Dart basics and practicing control flow, input/output, and logic.

---

## 📦 Requirements

- Dart SDK installed
